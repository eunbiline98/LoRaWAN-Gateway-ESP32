# LoRaWAN-Gateway-ESP32
RnD LoRaWAN with microcontroller ESP32
## List Item
### LoRa Gateway
- Esp32
- LoRA Module SX1276
- Antena Omni 8dBi
- Pigtail UFL to NF
- Ethernet Module W5500 Mini
- Display OLED **(optional)**

 ### LoRa Node
- Esp32
- LoRA Module SX1276
- Antena Omni 8dBi
- Pigtail UFL to NF
- Sensors and actuators can be provided, such as relays and DHT22
